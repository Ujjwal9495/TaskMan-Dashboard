# 🚀 TaskMan Dashboard

A modern and responsive task management dashboard built using React and Vite.

## ✨ Features

- 🌙 Dark Mode Toggle
- 📊 Statistics Cards
- 😊 Mood Tracker with Emojis
- ✅ Task Completion Animation
- 🎯 Priority-Based Tasks
- 💾 Local Storage Support
- 📱 Responsive Dashboard Layout
- 📂 Sidebar Navigation

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Ujjwal9495/TaskMan-Dashboard.git
```

Go into project folder:

```bash
cd TaskMan-Dashboard
```

Install dependencies:

```bash
npm install
```

Run project:

```bash
npm run dev
```
